# Fruit-Salad

This project is a advanced database using different methods about data storage and retrieval. It is designed to be a comprehensive solution for managing and querying large datasets efficiently.

The database supports various data types and provides powerful querying capabilities, making it suitable for a wide range of applications. With its robust architecture and optimized performance, Fruit-Salad is an ideal choice for developers looking for a reliable and scalable database solution. Whether you're building a small application or a large-scale enterprise system, Fruit-Salad offers the tools and features you need to manage your data effectively.

# Features

Inside Fruit-Salad, you will find a variety of features that make it a powerful and versatile database solution:

- **Efficient Data Storage**: Fruit-Salad uses advanced data storage techniques to ensure that your data is stored efficiently, reducing storage costs and improving performance.

- **Powerful Querying Capabilities**: With its robust querying engine, Fruit-Salad allows you to retrieve data quickly and easily, even from large datasets.

- **Views To See Most Important Data**: We uses three different views inside our database to show the most important types of data we had, which one:

    - **Total playtime played**: We have a view that shows the total playtime played by each user, this allows us to see how much time users are spending on our platform and identify our most active users.

    - **Achievement completation**: Using this view, we can see the total percentage of achievements completed by each user in our platform, this allows us to identify users who are highly engaged and motivated to complete every achievement.

    - **Total money spent**: This view shows the total amount of money spent by each user on our platform, we have different types of purchases inside our platform, such as game give aways, game purchases, and in-game purchases, this allows us to identify and display our most valuable users and understand their spending habits in every aspect of our platform.

- **Performace Optimization**: Fruit-Salad is designed with performance in mind, so you can be confident that your queries will run quickly and efficiently, even as your dataset grows.

### Features about this project

The database have a multiple tables to storage a lot types of data to make this project more efficient and organized, such as:

- **Users**: This table stores information about the users of our gaming platform, including their username, email, and registration date.
- **Games**: This table contains information about the games available on our platform, including the game title, genre, and release date.
- **Purchases**: This table records all the purchases made by users, including the games they have bought, the amount spent, and the date of purchase.
- **Playtime**: This table tracks the amount of time users spend playing each game, allowing us to analyze user engagement and identify popular games on our platform.
- **Achievements**: This table stores information about the achievements available in each game, as well as the achievements completed by users, allowing us to track user progress and engagement with the games on our platform.

With these tables and the powerful querying capabilities of Fruit-Salad, we can efficiently manage and analyze all the data related to our gaming platform, providing valuable insights into user behavior and preferences.
